Ajoute les cadenas Lockee à Wordpress.

== Description ==

Ce plugin permet de verrouiller le contenu des articles grâce à un cadenas Lockee (cadenas initialement disponible sur https://lockee.fr).

AVERTISSEMENT : Le type de cadenas et le code choisi sont enregistrés en clair dans la base de données de Wordpress.
